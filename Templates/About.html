About the "Google Play Store" dataset

Source of the Data

The data used in this project is sourced from the Google Play Store. It is a web-scraped dataset containing information about various Android apps.

Kaggle - Link to the dataset

Definition of Variables

  1. App Name: Application name
  2. Reviews: Number of reviews
  3. Size (mb): The size of the app in mb
  4. Installs (M): Number of app installs
  5. Type: This variable categorizes the app as "Paid" or "Free"
  6. Price in $: This column displays the price if the app is Paid
  7. Version: The current version of the application
